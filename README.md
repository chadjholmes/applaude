# Applaude

A beautiful desktop wrapper for the [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code) CLI.

Applaude provides a modern, intuitive interface for Claude Code, making it easier to manage sessions, track token usage, and visualize complex tasks.

## Features

- **Beautiful UI:** A clean, modern interface built with React and Tailwind CSS.
- **Session Management:** Save and resume your Claude Code sessions easily.
- **Process Management:** Seamlessly runs the Claude CLI in the background.
- **Real-time Streaming:** Native-feeling response streaming via `node-pty`.
- **Token & Cost Tracking:** Keep an eye on your context window and spending.

## Prerequisites

You must have the Claude Code CLI installed on your system:

```bash
npm install -g @anthropic-ai/claude-code
```

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run in development mode:**
   ```bash
   npm run dev
   ```

3. **Build the application:**
   ```bash
   npm run build
   ```
